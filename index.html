Por que usar frases-senha?

Uma frase-senha é composta por várias palavras aleatórias — como gato-vidro-nuvem-forte. Ela é ao mesmo tempo fácil de lembrar e difícil de adivinhar.

Senhas curtas e complicadas como P@ss1! são um pesadelo para lembrar e, ainda assim, fracas contra ataques automatizados. Cada palavra extra numa frase-senha multiplica o número de combinações possíveis: com este dicionário de 7.776 palavras, cada palavra adiciona cerca de 12,9 bits de entropia. Cinco palavras já ultrapassam 64 bits — mais forte do que a maioria das senhas tradicionais.

A ideia ficou famosa pela tirinha xkcd 936: «através de 20 anos de esforço, treinamos todo mundo a usar senhas difíceis para humanos lembrarem, mas fáceis para computadores adivinharem».
